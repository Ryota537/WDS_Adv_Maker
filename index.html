About Yume Stories

        
      

      

        
        
World Dai Star Scenario
          Maker

        

          A fan-made interactive visual scene generator & dialogue editor for the mobile game World Dai Star:
            Yume no Stellarium. Custom pose characters, set stage backgrounds, and write your own scripts!
        

        

          This project is open source and available on GitHub. If you have
          encountered any issues, please feel free to open an issue.
        

        

        

          Special thanks to Cpk0521 for the original WDS_Adv_Player base!
